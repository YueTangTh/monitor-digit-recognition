result_readme.txt

1. result data format:
	1) data: .m file(MATLAB) & .npy file(Python)
	2) image: extracted signal images

2. folder name:
	1) 'data' 
		1.1) 'MATLAB': .m files
		1.2) 'Python': .npy files
	2) 'image'
		extracted signal images

3. file name: 
	sheep number_video number_phase number
	for example, 3_11_2, 
		sheep 3, 
		video number 11(taken around 11 o'clock), 
		phase number 2
		
4. phase explanation:
	please see the image of 'phase.jpg'
